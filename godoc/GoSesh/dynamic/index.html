Overview ▹

Overview ▾

* * This module defines the Checker type, which provides a wrapper around GoVector capabilities. On top of GoVector's functions, it provides dynamic checking against a session type (usually defined by a mockup).

    type Checker

    type Checker struct {
    // contains filtered or unexported fields
}

    Stores the "current" session type in the computation. When network calls are made through the checker, they are checked against its type. It will panic if messages are of the wrong type, if sends and receives are mixed up or to the wrong party, or if sent labels are incorrect.

    func CreateChecker(id string, t multiparty.LocalType) Checker

    Create a checker with the given id (participant name) and (local) session type. GoVector logs are stored in ID_LogFile.txt, where ID is the value of id
